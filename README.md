# This is MMM v0.1:
Offline chat + file transfer over local network (Wifi/hotspot/same router)
No fantasy layers. JUst working system.

# Tech Stack (Simple & Real)
Node.js (runs on laptop + Android via Termux)
WebSocket (Real-time communication)
Local network (Wifi/hotspot)

# How to Run (REAL)

1. Install Node.js
Laptop: instal normally
phone: install via Termux

(Tiny version of node.js network app)

2. Install dependecies
npm install

3. Start server
npm run server

4. Start client (onsame or another device)
npm run client

5. Connect devices
Same Wifi OR
Phone hotspot

Replace on client:
YOUR-IP
